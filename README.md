Autókölcsönző Rendszer
Egyszerű objektumorientált Python projekt autók bérlésére, bérlésre és az aktuális bérlési listára.


Futtatás
bash python main.py


Megvalósított osztályok
•	Auto- absztrakt osztály
•	Szemelyauto- személyautók osztálya
•	Teherauto- teherautók osztálya
•	Autokolcsonzo- autók és bérlések kezelése
•	Berles- egy autó egy napra történő bérlése

Funkciók
•	Autók listázása
•	Autó bérlése egy napra
•	Bérlés lemondása
•	Aktuális bérlés listázása
•	Hibakezelés és adatvalidáció
•	Induláskor 3 autó és 4 bérlés előreése

Példa rendszerek
•	ABC-123
•	DEF-456
•	GHI-789
